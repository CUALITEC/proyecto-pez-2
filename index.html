Fase 4

Descripción

Importar librería

Para usar las librerías del servomotor y el ultrasónico primero se tienen que importar. Ésto añade nuevas funciones al programa desde las librerías.

Crear el objeto Servo

Para referirse a la librería servo, es necesario crear un nombre de esta librería en una variable (boca). Esto se conoce con el nombre de un objeto. Cuando se hace esto, se crea un nombre único que tendrá todas las funciones y capacidades que la librería servo posee. A partir de este punto en el programa, cada vez que se utilice el nombre "boca", se podrán usar todas esas funciones y capacidades de la librería servo.

Crear variables

Se necesitan crear las variables para referirse a los pines que estarán conectados al RGB y al ultrasónico, así como para referirnos a la distancia.


Declarar los pines

Dentro de la función setup() es necesario decirle a Arduino que los pines hacia el RGB son salidas. Además de esto se usa "boca.attach()" para indicar el pin conectado al servomotor.


Funcionamiento

Finalmente, en la función loop() se agrega la funcionalidad completa del pez. Se necesita que se cumpla lo siguiente:

Si la distancia es menor a 20, la boca se abre con un ángulo de 170° y el LED RGB se torna rojo.

Si no se cumple esa distancia, la boca se cierra con un ángulo de 50 y el LED RGB se torna verde.
